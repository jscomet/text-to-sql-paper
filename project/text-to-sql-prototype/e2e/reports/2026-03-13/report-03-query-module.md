## 发现的问题

### ISSUE-001: SQL 生成结果未显示

**严重级别**: High
**状态**: 待修复

#### 问题描述
点击"生成 SQL"或"一键运行"后，系统提示"SQL 生成成功"，但生成的 SQL 和查询结果未在页面上显示。

#### 期望行为
- 显示生成的 SQL 代码块
- 显示 SQL 执行结果表格
- 显示执行元信息（行数、时间）

#### 实际行为
- 仅显示成功提示
- 页面中间区域保持空白
- 查询历史未记录

#### 可能原因
1. 后端 LLM 服务未配置
2. API 响应格式问题
3. 前端状态管理问题

#### 建议修复
1. 检查后端 LLM 配置
2. 查看浏览器控制台网络请求响应
3. 验证前端组件渲染逻辑
