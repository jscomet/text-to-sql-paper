## 问题描述

点击"生成 SQL"或"一键运行"后，系统提示"SQL 生成成功"，但生成的 SQL 和查询结果未在页面上显示。

---

## 复现步骤

1. 登录系统，访问查询页面
2. 选择数据库连接
3. 输入自然语言问题，如"查询所有用户的信息"
4. 点击"生成 SQL"按钮
5. 观察提示和页面显示

---

## 期望行为

- 显示生成的 SQL 代码块
- 显示 SQL 执行结果表格
- 显示执行元信息（行数、时间）

---

## 实际行为

- 显示成功提示"SQL 生成成功"
- 页面中间区域保持空白
- 查询历史未记录

---

## 根本原因

前端状态管理或组件渲染逻辑问题，API 响应已正确返回，但前端未正确渲染结果。

---

## 修复状态

✅ **已修复并验证通过**

- SQL 生成成功，结果正确显示
- 显示生成的 SQL: `SELECT * FROM users;`
- 显示置信度: 90.0%
- 显示复制/执行按钮
