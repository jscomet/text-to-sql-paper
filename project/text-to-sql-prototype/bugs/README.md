## 严重级别说明

| 级别 | 图标 | 说明 |
|------|------|------|
| Critical | 🔴 | 系统崩溃、数据丢失、安全漏洞 |
| High | 🟠 | 主要功能不可用、阻塞测试执行 |
| Medium | 🟡 | 次要功能问题、有替代方案 |
| Low | 🟢 | UI/UX 问题、建议改进 |

## 状态说明

| 状态 | 图标 | 说明 |
|------|------|------|
| Open | 🟡 | 已发现，待修复 |
| In Progress | 🔵 | 修复中 |
| Fixed | 🟢 | 已修复，待验证 |
| Verified | ✅ | 已修复并验证通过 |
| Closed | ⚫ | 关闭（修复完成或不修复） |

## Bug 报告模板

新建 Bug 报告时请使用以下模板：

```markdown
# BUG-XXX: [标题]

## 基本信息

| 字段 | 内容 |
|------|------|
| **Bug ID** | BUG-XXX |
| **模块** | [模块名称] |
| **页面** | [路由] |
| **严重级别** | [🔴🟠🟡🟢] |
| **优先级** | [P0/P1/P2] |
| **发现日期** | YYYY-MM-DD |
| **发现人** | [姓名] |
| **状态** | [🟡/🔵/🟢/✅/⚫] |

## 问题描述

[简要描述问题]

## 复现步骤

1. [步骤1]
2. [步骤2]
3. [步骤3]

## 期望行为

- [ ] [期望1]
- [ ] [期望2]

## 实际行为

- [x] [实际1]
- [x] [实际2]

## 环境信息

| 项目 | 版本/地址 |
|------|----------|
| 前端 | http://localhost:5173 |
| 后端 | http://localhost:8000 |
| 浏览器 | [浏览器版本] |

## 建议修复方案

[描述建议的修复方案]

## 相关文档

- [链接1]()
- [链接2]()

---

*创建时间: YYYY-MM-DD*
*最后更新: YYYY-MM-DD*
*版本: v1.0*
```
