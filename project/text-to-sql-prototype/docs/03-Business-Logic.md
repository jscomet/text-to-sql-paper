### 3.1 SQLжӯЈзЎ®жҖ§йӘҢиҜҒз®—жі• (Execution-based Accuracy)

**з®—жі•жҸҸиҝ°пјҡ**
йҖҡиҝҮжү§иЎҢйў„жөӢSQLе’Ңй»„йҮ‘SQLпјҢеҜ№жҜ”жү§иЎҢз»“жһңйӣҶеҗҲжқҘеҲӨж–ӯжӯЈзЎ®жҖ§гҖӮ

**дјӘд»Јз Ғпјҡ**

```python
function verify_sql_correctness(pred_sql, gold_sql, db_file):
    """
    йӘҢиҜҒйў„жөӢSQLзҡ„жӯЈзЎ®жҖ§

    Args:
        pred_sql: жЁЎеһӢз”ҹжҲҗзҡ„SQL
        gold_sql: ж ҮеҮҶзӯ”жЎҲSQL
        db_file: ж•°жҚ®еә“ж–Үд»¶и·Ҝеҫ„

    Returns:
        correctness: 0 жҲ– 1 (1иЎЁзӨәжӯЈзЎ®)
    """
    conn = connect(db_file)
    cursor = conn.cursor()

    try:
        conn.execute("BEGIN TRANSACTION")

        # дҪҝз”ЁEXCEPTиҝӣиЎҢйӣҶеҗҲе·®иҝҗз®—
        sql_eq = """
            SELECT CASE
                WHEN EXISTS(SELECT 1 FROM (""" + pred_sql + """)
                            EXCEPT SELECT * FROM (""" + gold_sql + """))
                THEN 0
                WHEN EXISTS(SELECT 1 FROM (""" + gold_sql + """)
                            EXCEPT SELECT * FROM (""" + pred_sql + """))
                THEN 0
                ELSE 1
            END
        """

        cursor.execute(sql_eq)
        row = cursor.fetchone()
        correctness = int(row[0]) if row and row[0] is not None else 0

        conn.rollback()
        return correctness

    except Exception:
        conn.rollback()
        return 0
    finally:
        cursor.close()
```

- еҹәдәҺжү§иЎҢз»“жһңйӣҶеҗҲжҜ”иҫғпјҢиҖҢйқһSQLеӯ—з¬ҰдёІеҢ№й…Қ

- ж”ҜжҢҒеӨҡеҲ—з»“жһңгҖҒйҮҚеӨҚиЎҢгҖҒNULLеҖјзӯүеӨҚжқӮжғ…еҶө

### 3.2 еӨҡж•°жҠ•зҘЁз®—жі• (Majority Voting)

**з®—жі•жҸҸиҝ°пјҡ**
еҜ№еҗҢдёҖй—®йўҳйҮҮж ·еӨҡж¬Ўз”ҹжҲҗпјҢз»ҹи®Ўжү§иЎҢз»“жһңзҡ„еҮәзҺ°йў‘ж¬ЎпјҢйҖүжӢ©зҘЁж•°жңҖеӨҡзҡ„з»“жһңгҖӮ

**дјӘд»Јз Ғпјҡ**

```python
function majority_voting(pred_sqls, db_files, sampling_num):
    """
    еӨҡж•°жҠ•зҘЁйҖүжӢ©жңҖдҪіSQL

    Args:
        pred_sqls: жүҖжңүйҮҮж ·SQLеҲ—иЎЁ (flattened)
        db_files: еҜ№еә”зҡ„ж•°жҚ®еә“ж–Үд»¶еҲ—иЎЁ
        sampling_num: жҜҸдёӘй—®йўҳзҡ„йҮҮж ·ж¬Ўж•°

    Returns:
        mj_pred_sqls: жҠ•зҘЁеҗҺзҡ„SQLеҲ—иЎЁ
    """
    # 1. жү§иЎҢжүҖжңүйҮҮж ·SQL
    execution_results = []
    for i, (sql, db) in enumerate(zip(pred_sqls, db_files)):
        result = execute_sql(db, sql, timeout=30)
        execution_results.append({
            "data_idx": i,
            "sql": sql,
            "query_result": result.data if result.success else None,
            "valid": result.success
        })

    # 2. жҢүй—®йўҳеҲҶз»„иҝӣиЎҢжҠ•зҘЁ
    mj_pred_sqls = []
    for i from 0 to len(execution_results) step sampling_num:
        group = execution_results[i : i + sampling_num]

        # еҰӮжһңе…ЁйғЁж— ж•ҲпјҢиҝ”еӣһйҡҸжңәдёҖдёӘжҲ–й”ҷиҜҜж Үи®°
        if sum(r.valid for r in group) == 0:
            if return_random_when_all_errors:
                mj_pred_sqls.append(random_choice(group).sql)
            else:
                mj_pred_sqls.append("Error SQL")
            continue

        # з»ҹи®Ўеҗ„з»“жһңеҮәзҺ°йў‘ж¬Ў
        vote_count = {}
        for r in group:
            if r.valid:
                if r.query_result in vote_count:
                    vote_count[r.query_result]["votes"] += 1
                else:
                    vote_count[r.query_result] = {"votes": 1, "sql": r.sql}

        # йҖүжӢ©зҘЁж•°жңҖеӨҡзҡ„з»“жһң
        winner = max(vote_count.values(), key=lambda x: x["votes"])
        mj_pred_sqls.append(winner["sql"])

    return mj_pred_sqls
```

- еҸҜй…ҚзҪ®еҪ“е…ЁйғЁеӨұиҙҘж—¶зҡ„еӨ„зҗҶзӯ–з•Ҙ

### 3.3 Pass@K иҜ„дј°з®—жі•

**з®—жі•жҸҸиҝ°пјҡ**
Kж¬ЎйҮҮж ·дёӯеҸӘиҰҒжңүдёҖж¬ЎжӯЈзЎ®еҚіз®—йҖҡиҝҮгҖӮ

**дјӘд»Јз Ғпјҡ**

```python
function pass_at_k_evaluate(pred_results, gold_sqls, db_files, k):
    """
    Pass@KиҜ„дј°

    Args:
        pred_results: йў„жөӢз»“жһңеҲ—иЎЁпјҢжҜҸдёӘеҢ…еҗ«kдёӘйҮҮж ·
        gold_sqls: й»„йҮ‘SQLеҲ—иЎЁ
        db_files: ж•°жҚ®еә“ж–Үд»¶еҲ—иЎЁ
        k: йҮҮж ·ж¬Ўж•°

    Returns:
        accuracy: Pass@KеҮҶзЎ®зҺҮ
    """
    all_scores = []  # жҜҸйЎ№дёәдёҖж¬ЎйҮҮж ·зҡ„0/1еҲ—иЎЁ

    for sample_idx from 0 to k-1:
        # жҸҗеҸ–з¬¬sample_idxдёӘйҮҮж ·
        sample_sqls = [pred[sample_idx] for pred in pred_results]

        # иҜ„дј°иҜҘйҮҮж ·
        scores = []
        for pred, gold, db in zip(sample_sqls, gold_sqls, db_files):
            correctness = verify_sql_correctness(pred, gold, db)
            scores.append(correctness)

        all_scores.append(scores)

    # д»»дёҖйҮҮж ·йҖҡиҝҮеҚіз®—жӯЈзЎ®
    pass_at_k_scores = [
        1 if any(column) else 0
        for column in zip(*all_scores)
    ]

    return sum(pass_at_k_scores) / len(pass_at_k_scores)
```

```python
function find_foreign_key_candidates(db_path):
    """
    дҪҝз”ЁMinHashеҸ‘зҺ°жҪңеңЁеӨ–й”®е…ізі»

    жөҒзЁӢпјҡ
    1. дёәжҜҸеҲ—еҲӣе»әMinHashзӯҫеҗҚ (128дёӘhashеҮҪж•°)
    2. и®Ўз®—еҲ—й—ҙJaccardзӣёдјјеәҰ
    3. зӣёдјјеәҰ>0.85зҡ„и§ҶдёәеҖҷйҖүеӨ–й”®
    4. ж №жҚ®е”ҜдёҖеҖјж•°йҮҸжҺЁж–ӯж–№еҗ‘(FK->PK)
    """
    NUM_HASH_FUNCTIONS = 128
    SIMILARITY_THRESHOLD = 0.85

    # дёәжҜҸеҲ—еҲӣе»әMinHashзӯҫеҗҚ
    sketches = {}
    for (table, column) in get_all_columns(db_path):
        sketch = create_minhash_sketch(db_path, table, column)
        sketches[f"{table}.{column}"] = sketch

    # жҜ”иҫғжүҖжңүеҲ—еҜ№
    candidates = []
    for (field1, sketch1), (field2, sketch2) in combinations(sketches, 2):
        similarity = estimate_jaccard_similarity(sketch1, sketch2)

        if similarity >= SIMILARITY_THRESHOLD:
            # ж №жҚ®е”ҜдёҖеҖјж•°йҮҸжҺЁж–ӯж–№еҗ‘
            count1 = count_distinct(field1)
            count2 = count_distinct(field2)

            if count1 <= count2:
                candidates.append((field1, field2, similarity))
            else:
                candidates.append((field2, field1, similarity))

    return sorted(candidates, key=lambda x: x[2], reverse=True)

function create_minhash_sketch(db_path, table, column):
    """дёәеҲ—еҲӣе»әMinHashзӯҫеҗҚ"""
    sketch = [infinity] * NUM_HASH_FUNCTIONS

    for value in get_distinct_values(table, column, limit=10000):
        processed = str(value).lower().encode('utf-8')
        for i in range(NUM_HASH_FUNCTIONS):
            hash_val = mmh3.hash(processed, seed=i, signed=False)
            sketch[i] = min(sketch[i], hash_val)

    return sketch
```
